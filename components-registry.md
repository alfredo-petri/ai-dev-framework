# Components Registry

Catálogo leve de componentes criados ou refatorados com potencial de reutilização. Cada entrada deve ser registrada no mesmo changeset em que o componente é criado ou significativamente alterado.

## Propósito

- Facilitar descoberta e reuso de componentes
- Evitar duplicação acidental
- Servir como referência rápida para novos colaboradores e agentes de IA

## Template de entrada

```markdown
### ComponentName

- **Localização**: `path/to/ComponentName.tsx`
- **Tipo**: reutilizável | feature-local
- **Propósito**: Descrição de uma linha do que o componente faz
- **Responsabilidades**:
  - ...
- **Props principais**: `prop1: Tipo`, `prop2: Tipo`
- **Dependências**: `OutroComponente`, `useHookX`
- **Notas**: Contexto relevante, restrições, padrões visuais seguidos
```
